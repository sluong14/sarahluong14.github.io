# sarahluong14.github.io
IAT355 - Assignment 2 - Create A personal Website

## Purpose
this website is a personal portfolio created to showcase my background, skills, and interests in interactive arts and technology. It presents information similar to a resume while also demonstrating basic front-end web development and data visualization using HTML, CSS, and JavaScript.

The site is hosted using GitHub Pages and is intended to be a simple, professional online presence.

## Technologies Used
html: strucuturing content the website, CSS: for layout, styling, and responsive design, javascript for coding SVG visualizations, SVG: for data visualization and graphics, github: for hosting and pushing

## Site Structure & Navigation
The website consists of two main pages:

### Home (`index.html`)
- Personal introduction and overview  
- Education, skills, and experience sections  
- Contact form with basic client-side validation  
- Links to GitHub and LinkedIn  

### Visualizations (`visualizations.html`)
- An SVG bar chart visualization created using JavaScript  
- A creative SVG visualization created externally

Navigation between pages is handled through the header navigation bar, with a contact link that scrolls to the contact section on the homepage.

## JavaScript Files
main.js: Handles simple interactivity on the homepage, including contact form validation and small dynamic updates.
vis.js Responsible for generating and rendering SVG visualizations using JavaScript.
